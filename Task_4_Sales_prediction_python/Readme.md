# 📈 Sales Prediction using Python

## 🎯 Objective

Predict product sales based on advertising expenditure across different marketing platforms.

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 🔄 Project Steps

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Selection
* Linear Regression Model
* Model Evaluation
* Sales Prediction

## 🤖 Model Used

* Linear Regression

## 📊 Evaluation Metrics

* MAE
* MSE
* RMSE
* R² Score

 ✅ Conclusion

The Linear Regression model predicts product sales using advertising budgets.
TV advertising has the highest impact on sales, while radio also contributes significantly,
helping businesses optimize marketing strategies and improve sales forecasting.
